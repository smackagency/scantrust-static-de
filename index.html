So funktioniert es: Connected Packaging

1. Eindeutige IDs, die auf Produkte gedruckt oder aufgebracht werden

IDs, die als Etiketten angebracht oder direkt auf Produkte gedruckt, als RFID, NFC oder gelasert werden.

2. Zuordnen von Daten zu Produkt-IDs

Die Daten der Codes werden durch Scansysteme oder Smartphone-Apps überprüft und aktualisiert.

3. Professionelle Nutzer oder Konsumenten scannen den Code

QR Code oder RFID bzw. NFC wird innerhalb der Lieferkette, von Vertriebspartnern oder vom Endverbraucher gescannt.

4. Handeln auf der Grundlage von Daten zu Route-to-Market

Erhalten Sie Reports und Alerts zur Route-to-Market, zum Produktschutz und Engagement mit Endkunden
