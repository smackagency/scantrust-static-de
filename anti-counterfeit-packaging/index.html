Warum Scantrust Anti-counterfeiting QR Codes verwenden?

Sofortige Authentifizierung mit nur einem Smartphone

 

  • Scannen Sie den gesicherten QR Code zur Authentifizierung mit einer Handykamera

 

  • Mehrere Optionen zum Scannen, einschließlich einer speziellen mobilen App oder einfach über eine mobile Webseite

 

  • Kompatibel mit den meisten modernen Smartphones

 

  • Einbettung in Ihre eigenen Apps und Webseiten oder über API oder Weiterleitung

 

  • Authentifizierung auf Inspektorenebene mit Login wird ebenfalls unterstützt

Daten für die Fälschungsbekämpfung erhalten

 

Sehen Sie auf einen Blick, wo bestätigte und wahrscheinliche Fälschungen von Kunden gescannt werden. Nutzen Sie die Daten für die Durchsetzung von Maßnahmen.

 

  • Algorithmen zur Erkennung von Fälschungen verwenden serialisierte QR-Code-Scanpositionen, die Anzahl der Scans und den physischen Abstand zwischen den Scans

 

  • Berichte und Warnmeldungen verwenden eindeutige QR-Code-Seriennummern für detailliertere, detailliertere Daten

Wie man fälschungssichere QR Codes auf Produktverpackungen anbringt

1. Gesicherte QR Codes, die als Etiketten angebracht oder direkt auf die Verpackung gedruckt werden

Über unsere Druckpartner oder direkt mit Ihren Geräten und Substraten.

2. Die Codes werden vor der Verwendung sicher aktiviert

Nur aktivierte Codes können vom Endbenutzer authentifiziert werden.

3. Der Benutzer authentifiziert den Code mit seinem Telefon

Funktioniert für Verbraucher, professionelle Nutzer und Inspektoren.

4. Anzeige von Alerts und Reports über Fälschungen

Handeln Sie auf der Grundlage von Fälschungsdaten, die auf Scans im Markt oder von Inspektoren basieren.
